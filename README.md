# 🐍 Python: De Fundamentos à Análise de Dados

Repositório central de estudos em Python, cobrindo desde a lógica de programação básica até o desenvolvimento orientado a objetos e análise exploratória de dados.

## 📂 Estrutura do Repositório

* **01-Fundamentos:** Manipulação de strings, tipos numéricos e operações básicas.
* **02-Estruturas-Logicas:** Aplicação de condicionais (if/else) e laços de repetição (for/while).
* **03-Orientacao-a-Objetos:** Projeto **Sabor Express** — Sistema de gestão de restaurantes utilizando Herança, Polimorfismo e Decorators.
* **04-Analise-de-Dados:** Tratamento e limpeza de dados (Data Wrangling) com Pandas e NumPy, incluindo técnicas de imputação e conversão de tipos.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.12+
* **Bibliotecas:** Pandas, NumPy
* **Ferramentas:** VS Code, PyCharm, Google Colab e Git.
